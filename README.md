# Customer Loan Risk Prediction using Machine Learning

## Description
This project focuses on predicting customer loan risk using machine learning techniques. It involves data preprocessing, feature engineering, model building, and evaluation to identify potential high-risk borrowers. The goal is to support better decision-making in financial services through data-driven insights.

## Dataset
The dataset consists of multiple related tables containing customer financial information, loan history, and behavioral data used for risk assessment.

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
